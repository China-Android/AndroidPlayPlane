## 主要功能介绍:

1. 飞机自动向前飞行，手指拖动飞机即可进行飞机方向以及前后控制
2. 飞机自动发射子弹，击落敌机即可获取相应分数
3. 关卡设置，当分数到达指定分数，即可进入下一关，飞机速度增加，第几数量增加
